# core-utils

This is a shared NestJS library that contains utility functions and classes that can be used across different parts of the application.

## Building

Run `nx build core-utils` to build the library.

## Running unit tests

Run `nx test core-utils` to execute the unit tests via [Jest](https://jestjs.io).

## Structure

`src/lib`: Contains the source code for the library.

- `src/lib/utils`: Contains utility functions and classes.
- `src/lib/pipes`: Contains custom pipes used across the application.
- `src/lib/filters`: Contains custom exception filters used across the application.
- `src/lib/decorators`: Contains custom decorators used across the application.
- `src/lib/guards`: Contains custom guards used across the application.
